le jeu du pendu

Après 8 jours de découverte de Python, le but était de coder un jeu du pendu et d'y rajouter des fonctionnalités au choix.

Description du projet

Exercice final de la prépiscine, le programme demandait de reprendre l'ensemble des bases vu pendant la semaine. le jeu devait à minima prendre des mots dans un fichier externe puis lancer le jeu. Le jeu contient en plus un système de sauvegarde ainsi que 3 niveaux de difficulté.

Aperçu du projet

Une fois lancé, le jeu demande à l'utilisateur son niveau de difficulté choisi. Plus le niveau est élevé, moins l'utilisateur a de chance pour réussir. Ensuite le jeu affiche le nombre de lettre d'un mot pris aléatoirement dans le fichier mot_clé.txt. En cas d'erreur, le nombre de guess diminue de 1 si une seule lettre avait été proposée. Si c'etait un mot qui était proposé, le joueur perd 5 points d'un coup. le jeu s'arrête soit quand le joueur a épuisé toutes ses propositions ou s'il a réussi à trouver le mot. En cas de victoire, le jeu vérifie si le mot avait déjà été trouvé. Si le mot n'avait pas été trouvé ou si le mot a été trouvé en moins de proposition, le jeu propose de rentrer son nom pour enregistrer les highscore
