Job Board

Le but de ce projet était de simuler un site de recherche d'emploi.

Description du projet

Projet scolaire réalisé en deux semaines, le programme consistait à créer une application avec une base de données, une page web pour le front et une API, permettant de découvrir les concepts de Modèle-Vue-Contrôleur et la gestion d'une API.

Le groupe a choisi de travailler avec Node.js (frameworks: Express) pour la gestion du back en lien avec PostgreSQL pour la base de données SQL. Le front étant assez basique, il a été choisi de rester sur du HTML/CSS sans frameworks.

Aperçu du projet


Quand on se connecte, on arrive sur une page de login (puis vers un system d'insciption si la personne n'a pas de compte).

Une fois logué, l'utilisateur voit l'ensemble des offres disponibles (créé à partir d'un template) pouvant découvrir les détails de l'offre ou postuler (via des modales) les informations de l'utilisateur sont préremplies lorsqu'il postule et il peut à tout moment changer ses informations si besoin.

les compagnies ont une vue différente où elles peuvent voir les messages des personnes ayant postulé.

L'administrateur a lui aussi une vue dédiée où il peut réaliser un CRUD complet de toutes les tables.

Travail effectué

Dans ce projet, Thomas a réalisé plusieurs étapes :
- Organisation de la base de données,
- Création de la page applicant (html, css, js) ainsi que des routes, controllers et méthodes associées,
- Rédaction du README.md,
- Présentation du produit fini.
